Bitácora Digital

Bienvenid@ a mi portafolio virtual de Laboratorio 2. Este espacio ha sido creado con la finalidad de documentar mediante fotos y pequeñas descripciones acerca de las técnicas y conceptos que se aprendieron durante estos tres promedios los cuales conforman todo el ciclo a modo de recordatorio. Espero que esta plataforma te ayude a traer a memoria todo lo visto en clase!.

Semana 1

Durante esta primera semana lo que se hizo fue, a manera de introducción, reforzarnos temas vistos en Laboratorio 1, temas como el Arduino y sobre Datos de programación en general. También se nos enseñó un poco sobre cómo inspeccionar una página para leer los códigos de esta.

Semana 2

Durante esta semana se reforzó el tema de las páginas web, en esta ocasión nos enseñaron sobre cómo editar una plantilla HTML y como editar esta mediante VisualCode Se nos enseñó a cambiar las imágenes dentro de esta plantilla, a cambiar los colores, el texto, etc.

Semana 3

Durante esta semana se siguió viendo el tema de la edición de las plantillas en HTML debido a que vamos a trabajar con esto durante todo el ciclo para así recopilar toda la información vista en clase. Se vió también sobre Hosting y sobre el proceso de compra de dominios para así más adelante tener un dominio propio para nuestra página web.

Semana 4

Durante esta semana se nos enseñó acerca de Google Sites, sobre cómo personalizar diseños y crear una página de inicio. También sobre cómo interactuar con usuarios mediante las herramientas para agregar contenido.

Semana 5

Durante esta última semana de Promedio 1, se nos explicó nuestro proyecto final de promedio, el cual consistía en buscar una solución a una problemática haciendo uso de las ODS, la solución claramente tendría que ser tecnológica. Se nos pidió tambien, junto al proyecto, el crear un prototipo de baja o alta fidelidad o un wireframe para organizar y estructurar nuestra página web la cual tendrá que ser creada más adelante para Promedio 3.
